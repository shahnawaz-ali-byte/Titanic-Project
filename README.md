# 🚢 Titanic Survival Prediction

This project predicts whether a passenger survived or not using Machine Learning.

## 📊 Model Used
- Logistic Regression

## ⚙️ Technologies
- Python
- Pandas
- Scikit-learn

## 🧹 Data Preprocessing
- Filled missing values (Age, Embarked)
- Converted categorical data to numerical
- Dropped unnecessary columns

## 📈 Accuracy
- Achieved accuracy of **~77%**

## 📂 Files
- `Titanic_project.ipynb` → Main code
- `titanic.csv` → Dataset

## 🎯 Conclusion
The model successfully predicts survival with good accuracy using basic ML techniques.
